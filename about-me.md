# About Me — James Filmer

## Who I Am

My name is James Filmer. I'm a **Certified Financial Planner (CFP)** based in **Nashville, Tennessee**. I run my own financial planning practice and I'm building Eve as a side project — partly because I love storytelling and craft, and partly because I want to build something that becomes its own income stream independent of my practice.

I am a husband and a new dad (newborn daughter). My time is genuinely limited. I work on this project in whatever pockets of time I can find — sometimes 20 minutes, sometimes a few hours on a weekend. I need to be able to stop and pick up exactly where I left off without re-explaining everything.

I am a Christian. My values shape how I work and what I build. I want Eve to be something I'm proud of — honest, useful, and not built on manipulation or dark patterns.

## My Technical Reality

**I have zero coding experience.** I am not a developer. I do not know what most code means line by line. What I can do:

- Open Terminal and run commands I'm given
- Open VS Code, press Cmd+A, delete, and paste new code
- Follow step-by-step guides if they're written clearly
- Recognize when something is broken vs. working
- Spot when output doesn't match what was described

I learn by doing and by having things explained in plain English. I don't need to know *why* code works — I need to know *what to do* and *what to expect*.

## My Working Pattern

- I work in short sessions — 20 minutes to 2 hours
- I always come back to this project after breaks of hours or days
- I use Claude.ai Projects so context persists
- When something breaks, I copy the exact error message and ask for help
- I do NOT push code to GitHub until `npm run build` passes

## My Goals for This Project

**Short term:** Get Eve to a state where real users can sign up, create projects, and have their data persist. Working auth, working data layer, no crashes.

**Medium term:** Wire all data pages to Supabase, build Stripe Pro tier, get paying users. Build the features that make writers come back — session mode, timeline view, themes tracker.

**Long term:** Turn Eve into a real SaaS business. Writers who reject AI tools will pay for something that respects their craft. This is a backup income stream for a profession under real pressure from automation.

## My Relationship with AI

I use Claude as my developer. I don't resent this — I think it's the rational play. I'm not going to become a programmer with a newborn and a full-time practice. What I *can* do is direct, make product decisions, and deploy. That's the division of labor I've accepted. I trust Claude to write good code. I trust myself to know when something feels wrong or off-brand.

## Key Fact for Every Session

I am the only person building this. There is no team. Every decision runs through me.
