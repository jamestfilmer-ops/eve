EVE — BUILD REFERENCE
Last updated: Session 5
========================

WHAT EVE IS
-----------
A story organization and development workspace for writers.
No AI calls inside the app. No content generation.
Eve is a coach, organizer, and craft educator.
Claude builds the app — it does not run inside it.

TECH STACK
----------
Framework:   Next.js App Router (no src/, no TypeScript)
Language:    JavaScript only
Styling:     CSS variables in app/globals.css + Tailwind
Fonts:       Playfair Display (headings) + Source Sans 3 (body)
Colors:      Forest green (#2D5016) + white (#FAFCF7)
Database:    Supabase — schema exists, NOT YET WIRED to pages
Hosting:     Vercel — see docs/deploy-guide.txt to go live

CRITICAL CSS RULE
-----------------
Google Fonts @import MUST come BEFORE @import "tailwindcss" in globals.css.
Swapping the order causes a build error.
